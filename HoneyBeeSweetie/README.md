# 蜜花大作戰

## 1. 作品概要

- 制作者：周世熹、藍中璟、廣州tank、三溫暖教練
- 制作人数：チーム制作（本人は主にプログラム全般とゲーム企画を担当）
- 制作期間：2022年3月 / 約2か月
- 開発環境：C# / Unity / Visual Studio
- プレイリンク：http://hakka.ncu.edu.tw:8081/game
- ソースコード：現在確認中

台湾の国立中央大学のSDGs計画の一環として制作した、地域の養蜂文化を楽しく学べるウェブゲームです。Unityでのゲーム制作に挑戦したいと考え、チームに参加して制作しました。ゲーム性は横スクロールシューティングをベースにしており、Google Apps Scriptを使用したオンラインランキングも実装しました（現在は利用不可）。美術素材はチームメンバーが制作したものを使用しています。異なる専門分野のメンバーと協力しながら開発したことも、この作品で得た大きな学びです。

## 2. ゲーム内容

地域の養蜂文化を題材にした、横スクロールシューティング形式のウェブゲームです。プレイヤーはキャラクターを操作しながらステージを進み、遊びを通して蜂や地域文化に触れられるように設計しました。
