# ジャンケンマン

## 1. 作品概要

- 制作者：周世熹
- 制作人数：個人制作
- 制作期間：2026年2月 / 約1週間
- 開発環境：C++ / DirectXベースの学内簡易ゲームエンジン / Visual Studio

クラシックなゲーム「ジャンケンマン」を再現した作品です。基本的なじゃんけん機能に加えて、連勝数に応じた難易度調整や、最大当たり時のコイン噴出演出にも力を入れて制作しました。

## 2. ゲーム内容

画面上のスタートボタンを押すとゲームが始まり、中央に表示される相手の手が一定間隔で切り替わります。プレイヤーは下部のグー、チョキ、パーから手を選び、勝敗を判定します。

勝利すると連勝数が増え、相手の手が切り替わる速度が速くなります。最大連勝に到達すると、コインが画面上から大量に降る演出が発生します。負けると連勝数と速度がリセットされ、もう一度最初から挑戦できます。

## 3. 操作方法

- スタート：スタートボタンをクリック
- 手の選択：グー、チョキ、パーのボタンをクリック

## 4. ソースコードの構成

- `main.cpp` / `App.h` / `AppDef.h`：学内簡易ゲームエンジン側の基本構成とゲームループへの接続部分です。
- `App.cpp`：ゲーム全体の初期化、更新、描画、終了処理を各クラスへ渡します。
- `Logic.cpp` / `Logic.h`：じゃんけんの勝敗判定、連勝数、難易度変化を管理します。
- `Machine.cpp` / `Machine.h`：マウス入力の判定と、ゲーム進行の制御を担当します。
- `UI.cpp` / `UI.h`：背景、手の表示、勝敗表示、選択中エフェクトなどの描画を担当します。
- `CoinMaker.cpp` / `CoinMaker.h`：最大当たり時のコイン噴出演出を管理します。
- `GameDef.h`：じゃんけんの種類、勝敗、各UI座標などの共通定義をまとめています。

## 5. 見どころ・工夫した点

- 勝つほど相手の手の切り替え速度が上がるようにし、シンプルなルールの中に難易度変化を入れました。
- 最大連勝時にコインを大量に降らせることで、達成感のある演出を加えました。
- 勝敗判定、入力処理、UI表示、演出処理をクラスごとに分け、役割を整理しました。
