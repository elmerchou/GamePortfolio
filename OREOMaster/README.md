# OREO Master

## 1. 作品概要

- 制作者：周世熹
- 制作人数：個人制作
- 制作期間：2025年3月 / 約1か月半
- 開発環境：C# / Unity / Visual Studio

OREOをモチーフにしたタイピングゲームです。制限時間内に指定されたOREOの形を連続で組み立て、ハイスコアを目指します。コンボ数やスコアに応じて、制限時間の加算やスコアボーナスなどの報酬が得られるように設計しました。画像素材は、インターネット上のフリー素材とAI生成素材を組み合わせて使用しています。

## 2. ゲーム内容

画面に表示されるOREOの並びを確認し、キーボードで対応する文字を入力して同じ形を作るタイピングゲームです。正しく入力できるとスコアが加算され、コンボ数が増加します。

コンボ数が増えると、次に出題されるOREOの数が増え、難易度が上がります。また、一定コンボごとに制限時間が加算されます。入力が失敗するとコンボが途切れ、評価表示やアニメーションで結果を伝えるようにしています。

## 3. 操作方法

- 文字入力：キーボード
- 入力確定：Enterキー
- ゲーム開始・画面遷移：画面上のボタンをクリック

## 4. ソースコードの構成

- `eventsystem.cs`：入力内容の判定、OREO生成、スコア・コンボ・タイマーとの連携、ゲーム開始・終了演出を管理します。
- `oreo_generator.cs`：コンボ数に応じて出題するOREOの形を生成します。
- `combo.cs`：コンボ表示、コンボ数に応じた炎エフェクトの切り替えを担当します。
- `score.cs` / `scoreRecorder.cs`：スコア計算とシーン間のスコア保持を担当します。
- `Countdown.cs`：制限時間の管理、時間切れ時のゲーム終了処理を担当します。
- `grade.cs`：入力結果の評価表示とフェードアウト演出を担当します。
- `enemy.cs` / `oreo_attack.cs` / `enemy_cookie.cs`：入力成功時の攻撃・敵演出を担当します。
- `title.cs` / `Ending.cs` / `help.cs` / `back.cs`：タイトル、ヘルプ、エンディング、画面遷移を管理します。
- `SoundManager.cs`：BGMや効果音の再生を管理します。

## 5. 見どころ・工夫した点

- OREOの形を文字入力と結びつけ、見た目と入力ルールが直感的につながるようにしました。
- コンボ数に応じて出題量、スコア、時間加算、炎エフェクトが変化するようにし、継続して成功するほど気持ちよくなる設計にしました。
- 入力成功時にOREOが飛んでいく演出や敵の吹き飛び演出を入れ、タイピング結果が視覚的に分かるようにしました。
- タイトル、ゲーム本編、リザルト画面にフェード演出を入れ、画面遷移が自然になるようにしました。
