# [OREO Master](./OREOMaster)

## プレイ動画

https://github.com/user-attachments/assets/5418a6eb-adf6-42ea-9e5b-7459f3d9711b

## 概要

- 制作者：周世熹
- 制作人数：個人制作
- 制作期間：2025年3月 / 約1か月半
- 開発環境：C# / Unity / Visual Studio

OREOをモチーフにしたタイピングゲームです。制限時間内に指定されたOREOの形を連続で組み立て、ハイスコアを目指します。コンボ数やスコアに応じて、制限時間の加算やスコアボーナスなどの報酬が得られるように設計しました。画像素材は、インターネット上のフリー素材とAI生成素材を組み合わせて使用しています。

# [蜜花大作戰](./HoneyBeeSweetie)

## プレイ動画

https://github.com/user-attachments/assets/14313e95-2793-41f4-b5dd-cad1e56f9d4f

## プレイリンク

http://hakka.ncu.edu.tw:8081/game

## 概要

- 制作者：周世熹、藍中璟、廣州tank、三溫暖教練
- 制作人数：チーム制作（本人は主にプログラム全般とゲーム企画を担当）
- 制作期間：2022年3月 / 約2か月
- 開発環境：C# / Unity / Visual Studio
- ソースコード：現在確認中

台湾の国立中央大学のSDGs計画の一環として制作した、地域の養蜂文化を楽しく学べるウェブゲームです。Unityでのゲーム制作に挑戦したいと考え、チームに参加して制作しました。ゲーム性は横スクロールシューティングをベースにしており、Google Apps Scriptを使用したオンラインランキングも実装しました（現在は利用不可）。美術素材はチームメンバーが制作したものを使用しています。異なる専門分野のメンバーと協力しながら開発したことも、この作品で得た大きな学びです。
