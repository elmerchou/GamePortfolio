# 倉庫番

## 1. 作品概要

- 制作者：周世熹
- 制作人数：個人制作
- 制作期間：2026年3月 / 約1か月
- 開発環境：C++ / DirectXベースの学内簡易ゲームエンジン / Visual Studio

クラシックなゲーム「倉庫番」を再現した作品です。基本的な箱押しパズルのルールに加えて、ステージ選択、ステージリセット、一手戻しなどの機能を実装しました。

## 2. ゲーム内容

プレイヤーを操作して箱を押し、すべての箱をゴール位置へ運ぶパズルゲームです。壁や箱の位置を考えながら移動し、詰まないように手順を組み立てます。

ステージは外部データから読み込み、ステージ選択画面から遊ぶステージを選べます。プレイ中はステージリセットや一手戻しを使うことができ、試行錯誤しながら攻略できるようにしました。

## 3. 操作方法

- 移動：矢印キー
- ステージリセット：Spaceキー
- 一手戻し：Backspaceキー
- ステージ選択：上下キー
- 決定：Enterキー
- タイトルへ戻る：Escキー

## 4. ソースコードの構成

- `main.cpp` / `App.h` / `AppDef.h`：学内簡易ゲームエンジン側の基本構成とゲームループへの接続部分です。
- `GameObjectMng.cpp` / `GameObjectMng.h`：ゲーム中のプレイヤー、箱、マップ、クリア判定、ステージ読み込みを管理します。
- `Player.cpp` / `Player.h`：プレイヤーの移動、向き、アニメーション、移動可能判定を担当します。
- `Box.cpp` / `Box.h`：箱の移動、ゴール上にある場合の表示切り替えを担当します。
- `Tilemap.cpp` / `Tilemap.h`：マップデータの読み込み、壁・ゴール・箱・プレイヤー初期位置の管理を担当します。
- `Data/GamePlayData`：現在のステージやステージ開放状況を管理します。
- `Data/UndoData`：一手戻し用にプレイヤー位置、向き、箱の位置を保存します。
- `Scene`：タイトル、ステージ選択、ゲーム本編、ゲームクリアなどのシーン管理を担当します。
- `SE/SEManager`：移動、リトライ、一手戻しなどの効果音を管理します。

## 5. 見どころ・工夫した点

- 外部データからステージを作成することで、ステージ追加がしやすい構成にしました。
- 一手戻し機能を実装し、プレイヤーが試行錯誤しやすいようにしました。
- シーン管理を分け、タイトル、ステージ選択、ゲーム本編、クリア画面の流れを整理しました。
- プレイヤーと箱をタイル単位で移動させ、倉庫番らしい盤面パズルの動きを再現しました。
